Hardware

Despite it being the size of a credit card, the RPi packs a lot into what just looks like a motherboard. In this section, we explore the hardware architecture of the Raspberry Pi and point out some of the benefits and some of the downsides to having this cheap but effective device.

CPU (ARM)

The Raspberry Pi consists of a 700Mhz ARM11 ARM1176JZF-S core (32-bit RISC). This CPU is commonly used as the microprocessor for various pocket devices such as cell phones and MP3 players. In the past few years, the price of the ARM11 series has dropped significantly, making it the right choice to add to this cheap computer. This processor is a significantly improves piplelining compared to its predecessors . The newly designed pipe allows for load/store and ALU parellelism as well as dynamic branch predicting. ARM11 also increase the pipe from 5 stages to 8, which with correct branch prediction greatly increases throughput. 

GPU

The choice for a GPU was simple. Co-creator Eben Upton had already worked on the architecture team for the Broadcom VideoCore IV, making it easy to expand the GPU's features. The core uses OpenGL ES2.0 and OpenVG libraries in order for fast access to the core. The libraries perform at 24 GFLOPS (24 billion floating point operations per second). This gives RPi the capability of extremely good visual playback with a low amount of power. The Raspberry Pi Foundation equates the performance of the graphics to the first generation XBOX. The first released model of RPi (Model A) allocated 128 megabytes to the GPU. In order to improve performance, Model B allocated 256MB to the GPU. 

 

RAM

Multiple different mobile memory chips have been used for RPi since it was released. The latest chip, the 512MB Samsung Mobile DRAM, allows users to change how the RAM is split between the GPU and CPU. The increase in memory from the previous model increased the amount of power needed for the device.
