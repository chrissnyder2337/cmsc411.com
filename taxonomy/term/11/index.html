Glossary

Clock rate : a measure of how many clock cycles a CPU completes per unit of time. Overclocking : the process of configuring your CPU to run at a higher clock rate than the default set by the manufacturer. CPU multiplier : the multiplier that is applied to the front-side bus frequency to obtain the CPU clock rate. Front-side bus frequency : the frequency at which the motherboard’s front-side bus runs. This can cause the CPU to produce incorrect results for some percentage of its calculations. In some cases, this can cause the computer to refuse to start up. In other cases, the computer is able to boot, but calculation errors still occur intermittently, which can lead to unexpected crashes. The overclocking process

Overclocking is a process of trial and error. The goal is to find a balance point that maximizes your CPU’s clock rate while keeping the voltage at an acceptable level. The higher the clock rate, the higher the voltage your CPU requires. If the voltage is too low for the chosen clock rate, the CPU may start to produce erroneous results for some calculations, which can cause the computer to refuse to boot in some cases, or lead to transient crashes in others. What is overclocking?

CPU overclocking is the process of configuring your CPU to run at a higher clock rate than the default set by the manufacturer. This lets you get more bang for your buck in terms of processor speed and performance. So if a CPU has the potential to run at a higher clock rate, why doesn’t the manufacturer set it to run at the higher rate by default? That's because overclocking comes with a set of tradeoffs.
