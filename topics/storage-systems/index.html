Block Virtualization

The Basics Block virtualization is the separation of logical and physical storage so that the logical storage can be accessed regardless of the structure of the physical storage. In a block level storage device, raw storage volumes are created, and then the server-based operating system connects to these volumes and uses them as individual hard drives. This makes block virtualization extrememly flexible and versatile, being able to be used for almost any kind of application.
