Hypervisors

A Hypervisor or Virtual Machine Monitor is a piece of software, firmware, or hardware that creates and runs system virtual machines. A computer on which a hypervisor is running is called a Host Machine and the several system virtual machines (OSs) managed by the hypervisor are called Guest Machines. A hypervisor allows each guest OS to execute as though it owns the host's processor, memory, and other hardware resources (when in reality those resources are shared among all guest OSs). GPU Architecture

A graphics processing unit (GPU) is hardware that is designed to quickly manipulate memory in order to create images to drive the output of a display. GPUs are ubiquitous in modern computing; They exist in desktop computers, laptops, smartphones, game consoles, servers, and more. The purpose of a GPU was initially to drive displays, but their purpose has slightly evolved. Today, GPUs are mainly used for computer graphics and parallel computing due to their ability to process many blocks of data simultaneously.
