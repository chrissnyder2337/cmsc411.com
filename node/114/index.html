Console Comparison

 

 
Launch Price: $270 $400 $250
CPU Brand/Model: Cell Broadband Engine PowerPC Xenon IBM Broadway
CPU Clock Speed: 3.2 GHz  3.2 GHz 729 MHz
CPU Cores: 1 main, 8 single-threaded coprocessors (6 accessible) 3 dual-threaded  1 w/ 5 execution units
System FLOPS: 2 TFLOPS 355 GFLOPS 61 GFLOPS
RAM:

CPU - 256MB @ 700MHz

GPU - 256MB @ 3.2GHz

 Combined - 512MB @ 700MHz

Combined - 64MB

GPU - additional 3 MB 

 

Clearly, the Wii is not a contendor. But, which is better? The PS3 or the XBOX360?

Turns out, it depends. They're both better than the other at different tasks, as all of these hardware features come with tradeoffs. For example, half of the PS3's RAM is faster than the 360, but 360's RAM is shared between the CPU and GPU. Additionally, the 360's FLOPS is lower than the PS3, but the 360's architecture has more pipelines for things like shaders, and stores textures really effeciently, requiring less memory and bandwith. At the end of the day, they're both really powerful. 

If you want to read more, this post details their differences well: http://xboxrepublic.top-forum.net/t4734-xbox-360-hardware-vs-ps3-hardware-a-detailed-comparison
