[PS2] Architecture Overview

I/O Processor
The 34 mhz MIPS CPU(R3000) acts as the emulation computer to run game for earlier games,It is used to access the sound processing unit, controllers, cd and harddrive, as well as USB and fireWire port. I/O P has 2MB of memory, while the Sound processing unit has 2MB of memory.
 
Graphics synthesizer
Takes rendering command from emotion engine in a display lists format. (commands): lists of 32-bit command that tells the renderer what shape and where as well as what color and texture to fill them. It supports the basic features of rastering polygons, texturing, alpha blending, basic scissoring, zbuffer, etc. It does not support clipping.
 
Emotion Engine
Provides a connection between main memory, RAM, I/O  Processor, and graphic synthesizer.
 
Speed Type
Parallelism Exploitation. (FPU, 300 MHZ CPU, VPU0 and VPU1 can all work in parallel, all VPUs are SIMD, each VPU can perform 4 op in parallel)
